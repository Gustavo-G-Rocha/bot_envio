REQUISITOS OBRIGATÓRIOS:
========================

Você PRECISA ter Node.js instalado em seu computador.

Se ainda não tiver:
  1. Acesse: https://nodejs.org/
  2. Faça download da versão LTS (recomendada)
  3. Execute o instalador e siga as instruções
  4. Reinicie seu computador após a instalação

COMO USAR:
==========

1. Abra este arquivo: double-click em "build.bat"
   
   OU pelo terminal:
   - Abra o Prompt de Comando (cmd)
   - Digite: cd caminho_da_pasta
   - Digite: build.bat

2. Na primeira vez, o script instalará automaticamente 
   todas as dependências (pode demorar alguns minutos)

3. Você verá um menu com opções:

   Opção 1 - INICIAR A APLICAÇÃO
   ==============================
   Abre a aplicação pronta para usar
   Escaneie o QR Code com seu WhatsApp
   
   Opção 2 e 3 - COMPILAR PARA EXECUTÁVEL
   =======================================
   Cria um programa .exe independente
   Você pode enviar para outras pessoas
   
   Opção 4 - REINSTALAR DEPENDÊNCIAS
   ===================================
   Use se houver algum erro

DÚVIDAS?
========

Node.js não instala?
  - Tente baixar a versão LTS em: https://nodejs.org/
  - Escolha o instalador para seu Windows (32-bit ou 64-bit)

Script não funciona?
  - Certifique-se de que Node.js está instalado
  - Abra um Prompt de Comando e digite: node --version
  - Se aparecer uma versão, Node.js está instalado

Precisa ajuda?
  - Verifique se todos os arquivos foram copiados corretamente
  - Teste novamente executando o build.bat
