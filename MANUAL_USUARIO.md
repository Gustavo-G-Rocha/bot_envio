# 📱 Bot Envio WhatsApp - Manual do Usuário

## 🚀 Como Instalar e Usar

### Instalação
1. Baixe o arquivo `Bot Envio WhatsApp Setup X.X.X.exe`
2. Execute o instalador
3. Siga as instruções na tela:
   - Escolha onde instalar (ou use o local padrão)
   - Marque se deseja criar atalho na área de trabalho
4. Clique em "Instalar" e aguarde
5. Clique em "Concluir"

### Primeiro Acesso
1. Abra o aplicativo pelo atalho criado
2. Uma janela será aberta automaticamente
3. Aguarde alguns segundos para o app carregar

## 📋 Requisitos

- ✅ Windows 10 ou superior (64 bits)
- ✅ Conexão com a internet
- ✅ **NÃO precisa instalar Node.js, Python ou qualquer outra coisa!**

## 🎯 Primeiro Uso

1. **Abra o aplicativo** - Use o atalho criado na instalação
2. **Escaneie o QR Code** - Use seu WhatsApp para escanear
3. **Aguarde a conexão** - Levará alguns segundos
4. **Pronto!** Agora você pode enviar mensagens

## 🔐 Segurança

- ✅ Seus dados ficam apenas no seu computador
- ✅ A autenticação é feita diretamente com o WhatsApp
- ✅ Nenhuma informação é enviada para servidores externos

## 📂 Arquivos Criados

O aplicativo cria automaticamente estas pastas no diretório de instalação:

```
📂 .wwebjs_auth    → Suas sessões do WhatsApp
📂 .wwebjs_cache   → Cache temporário
📂 uploads         → Arquivos enviados
```

**⚠️ NÃO DELETE essas pastas** ou você precisará escanear o QR Code novamente!

## ❓ Problemas Comuns

### O QR Code não aparece
- Aguarde 10-20 segundos
- Feche e abra o aplicativo novamente

### Desconectou do WhatsApp
- É normal após um tempo sem usar
- Basta escanear o QR Code novamente

### Erro ao enviar mensagens
- Verifique sua conexão com internet
- Certifique-se que o WhatsApp está ativo no celular

## 🗑️ Desinstalar

1. Vá em **Configurações → Aplicativos**
2. Procure por **"Bot Envio WhatsApp"**
3. Clique em **Desinstalar**
4. Siga as instruções na tela

**Nota:** A desinstalação remove automaticamente os dados do aplicativo.
